## Project Overview

This project demonstrates the implementation of an Image Classification system using Deep Learning techniques. A Convolutional Neural Network (CNN) model was developed and trained to classify images into different categories using the CIFAR-10 dataset.

The project aims to understand the fundamentals of deep learning, image preprocessing, model building, training, and evaluation.

---

## Objectives

* Understand the basics of Deep Learning
* Build a CNN model for image classification
* Train the model using image datasets
* Evaluate model performance
* Generate predictions for image categories

---

## Dataset Information

**Dataset Used:** CIFAR-10 Dataset

The CIFAR-10 dataset contains 60,000 color images divided into 10 classes:

* Airplane
* Automobile
* Bird
* Cat
* Deer
* Dog
* Frog
* Horse
* Ship
* Truck

Dataset Details:

* Training Images: 50,000
* Testing Images: 10,000
* Image Size: 32 × 32 pixels
